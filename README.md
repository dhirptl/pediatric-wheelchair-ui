# Pediatric Power Wheelchair Navigation UI 🦽🎮

## Overview
[cite_start]This project addresses the gap in intuitive controls by developing a Unity-based gamified interface that integrates directly with a ROS 2 autonomous wheelchair system. [cite: 6] [cite_start]The goal is to enhance user engagement while maintaining navigation safety and realism for pediatric users. [cite: 9]

## Key Features
* [cite_start]**Dual Control Modes:** Supports both low-level motion control (virtual joystick) and destination-based autonomous navigation. [cite: 8, 17, 18]
* [cite_start]**Pediatric-Friendly UI:** Features a high-contrast color palette (e.g., yellow on black) to assist children with visual challenges. [cite: 25]
* [cite_start]**Gamification:** Includes collectible items, progress indicators, and visual feedback triggered by the wheelchair's actual pose rather than commanded motion. [cite: 24]
* [cite_start]**ROS 2 Integration:** Communicates with the Nav2 stack via standard ROS topics and action interfaces. [cite: 12, 15]

## Tech Stack
* [cite_start]**Engine:** Unity 2022.3 LTS (URP) [cite: 13, 29]
* [cite_start]**Robotics Framework:** ROS 2 (Humble)
