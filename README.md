# Pediatric Power Wheelchair Navigation UI 🦽🎮

## Overview
This project addresses the gap in intuitive controls by developing a Unity-based gamified interface that integrates directly with a ROS 2 autonomous wheelchair system. The goal is to enhance user engagement while maintaining navigation safety and realism for pediatric users.

## Key Features
* **Dual Control Modes:** Supports both low-level motion control (virtual joystick) and destination-based autonomous navigation.
* **Pediatric-Friendly UI:** Features a high-contrast color palette (e.g., yellow on black) to assist children with visual challenges.
* **Gamification:** Includes collectible items, progress indicators, and visual feedback triggered by the wheelchair's actual pose rather than commanded motion.
* **ROS 2 Integration:** Communicates with the Nav2 stack via standard ROS topics and action interfaces.

## Tech Stack
* **Engine:** Unity 2022.3 LTS (URP)
* **Robotics Framework:** ROS 2 (Humble)

## Project Context
This project is being developed as part of my research work in the NCBL (Neuromechanics Lab) at the University of Alberta. It is being built under the guidance of a graduate student researcher to directly support ongoing academic work in assistive pediatric robotics.
